# Fixer Upper

  Fixer Upper is designed to help home owners keep track of all the costs and details of their home improvement projects.

## Description

  With Fixer Upper, users will be able to create projects for all their home repairs. Their projects can be a wish list for the next five years, an on going checklist of   completed projects or projects currently in progress. 
  
  Users have the ability to add, delete and edit contractor information, estimates, expenses and general notes about each project. Fixer Upper also gives users a           contractor contact list for saving their favorite contractors. This list can also be added to, edited or deleted.
  
## Technology

  Technologies used: HTML, CSS, Javascript, and React JS
  
  React packages include: react-icons, react-scripts, react-select and reactstrap
